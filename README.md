# Phonebook Backend

This is the backend for the Phonebook application.

## 🌐 Live Application

👉https://three-11-full-stack-phonebook.onrender.com

## 📡 API Endpoints

- GET /api/persons
- GET /api/persons/:id
- POST /api/persons
- DELETE /api/persons/:id

## 🛠 Tech Stack

- Node.js
- Express
- Morgan (logging)
- CORS
